## What I'm working on

**Out-of-Order RISC-V Processor** &nbsp;·&nbsp; Verilog  
A superscalar OOO processor using Tomasulo's algorithm with a Register Alias Table and Reorder Buffer. Synthesised on Artix-7.

**CDC Verification Framework** &nbsp;·&nbsp; SystemVerilog, VCS, SpyGlass  
A full three-layer verification environment for clock domain crossings — static analysis, SVA monitors, and functional coverage. Hit 100% coverage and clean sign-off.

**Real-Time Task Offloader** &nbsp;·&nbsp; C, Linux, STM32  
A Linux character driver and STM32 firmware that offloads latency-critical tasks from the host. Got jitter under 20 µs by bypassing TTY entirely with a custom IOCTL.

**Continuum Robot RL** &nbsp;·&nbsp; Python, PyTorch  
Physics-based sim environment for controlling a continuum robot using PPO. Still a work in progress.
